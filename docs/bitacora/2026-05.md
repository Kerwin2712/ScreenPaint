# Bitácora de Cambios - Mayo 2026

## Resumen del Mes
Este mes se centra en la experiencia de usuario (UX) y la fluidez de las herramientas de anotación, comenzando con una reinvención total de la herramienta de texto.

---

### [2026-05-16] - Revolución del Texto In-Place
*   **Eliminación de Diálogos:** Se eliminó la ventana emergente `TextDialog` para la creación de texto.
*   **Edición In-Place:** Ahora el texto se escribe directamente sobre el lienzo al hacer clic, proporcionando una experiencia tipo "Canvas" o "PowerPoint".
*   **Auto-redimensionamiento:** Los cuadros de texto ahora calculan su tamaño dinámicamente según el contenido escrito.
*   **Edición de Existentes:** Se habilitó la capacidad de re-editar textos previos simplemente haciendo clic sobre ellos con la herramienta de mano.
*   **Atajos de Ergonomía:**
    *   `Ctrl + Enter` para finalizar la edición.
    *   Foco automático al crear un nuevo texto.
*   **Correcciones Técnicas:**
    *   Solucionado error de tipo en el manejo de colores (Enum vs String).
    *   Solucionado error de tipo en el redimensionamiento del documento (`QSizeF` vs `QPointF`).
*   **Documentación:**
    *   Creación de la carpeta `docs/bitacora` para el registro histórico y futuro del proyecto.
    *   Resumen detallado de commits anteriores organizado por meses.
*   **Bola Flotante de Opciones (UI):**
    *   Creación de un panel dinámico para ajustar el texto sin diálogos.
    *   Control de tamaño de fuente (+/-).
    *   Paleta de colores expandida: Rojo, Amarillo, Cian, **Blanco y Negro**.
    *   **Herramienta de Movimiento:** Adición de un "Grip" (✥) que permite desplazar el cuadro de texto libremente mientras se edita.
*   **Correcciones de Estabilidad (Hotfixes):**
    *   Restaurado el método `_get_point_at` (colisión de puntos) eliminado por error.
    *   **Bloqueo de Atajos:** Se desactivaron los atajos de teclado globales (como la 'T' o el borrador) mientras el editor de texto está activo, permitiendo el uso normal de mayúsculas y escritura fluida.
*   **Menú Circular de Colores (UI):**
    *   Reemplazo del diálogo de color estándar por un menú circular dinámico (`CircularColorMenu`).
    *   Muestra 5 colores rápidos (Rojo, Amarillo, Cian, Blanco, Negro) alrededor del puntero.
    *   Incluye un acceso directo a las "Opciones Avanzadas" (diálogo tradicional) mediante un icono de engranaje.
*   **Gestión de Versiones:**
    *   Sincronización continua con el repositorio de GitHub.
